# samples

Git Large File Storage 限制了总的文件体积，无法上传样本了，该仓库封存。

样本越来越难下载了，建一个仓库，放样本和相关描述。  

结构：  
```
samples
    <sha256>.7z, 密码: infected
sample_info
    <sha256_folder>/<normal_name_folder>
        images_dir
        files_dir, 不能存会报毒的文件
        readme.md
    ...
README.md
```

需要新建的文件夹名就是 `images` 和 `files`
